# STL Sprue Generator

Веб-приложение для автоматического разбиения STL-моделей на детали и генерации спрусов для литникового литья.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
uvicorn backend.main:app --reload
```

Откройте браузер по адресу http://localhost:8000

## Использование

1. Загрузите STL файл через веб-интерфейс
2. Приложение автоматически предложит варианты разбиения модели на части
3. При необходимости скорректируйте границы раздела
4. Сгенерируйте спрусы и экспортируйте детали в отдельные STL файлы
